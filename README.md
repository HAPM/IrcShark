<h1>IrcShark</h1>
<p>
IrcShark ist ein Programmunabhängiger IRC Client / Bot der in C# / Mono geschrieben wurde und eine hohe Erweiterbarkeit und Portabilität bietet.<br />
Er befindet sich derzeit in einer frühen Entwicklungsphase, daher suchen wir noch findige Menschen die Lust und Zeit haben uns zu unterstützen.
</p>
